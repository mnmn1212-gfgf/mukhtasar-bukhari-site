مختصر البخاري - نسخة مطوّرة

تشغيل المشروع:
1) افتح المجلد في VS Code أو Terminal.
2) نفّذ:
   npm config set registry https://registry.npmjs.org/
   npm install
   npm run dev

أهم ملفات التعديل:
- src/main.jsx    المحتوى والصفحات والبيانات
- src/styles.css  الألوان والتنسيق والهوية البصرية

ملاحظات:
- المحتوى المعروض داخل الأحاديث مختصر ومخصص للعرض الأولي.
- عند إدخال كامل مادة الكتاب يجب مراجعة نصوص الأحاديث والكتب والأبواب مراجعة علمية دقيقة قبل النشر النهائي.
